🤖 AI Robot Factory
Concepts Used
Inheritance
Method Overriding
Classes & Objects
Polymorphism (basic level)


Description

A smart AI Robot Factory system where different types of robots inherit from a common Robot class and perform unique tasks based on their specialization.

Each robot shares common properties like name and battery level, but behaves differently based on its role.

Features
Cleaning Robot performs floor cleaning tasks 🧹
Security Robot patrols and detects intruders 🛡️
Delivery Robot delivers packages 📦
Check status of all robots
Charge all robots to full battery 🔋
Interactive menu-driven system

How It Works
A base class Robot contains common properties and methods
Child classes extend Robot and override performTask()
Each robot has its own specialized behavior
Demonstrates real-world inheritance structure

How to Run
Open project in Eclipse IDE  
2. Navigate to `Main.java` inside `com.secure.ewallet` package
