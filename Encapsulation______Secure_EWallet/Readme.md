# Secure E-Wallet 💳

## Concepts Used
- Encapsulation
- Private variables
- Getters/Setters
- Validation

## Description
A secure wallet system where balance and PIN are protected using encapsulation.

## Features
- Deposit money
- Withdraw money with PIN
- Check balance securely
- Transaction history

## How to Run
1. Open in Eclipse
2. Run Main.java
