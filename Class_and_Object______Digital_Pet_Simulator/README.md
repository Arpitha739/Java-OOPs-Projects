# Digital Pet Simulator 🐶🐱🐉

## Concepts Used
- Classes & Objects
- Constructors
- Methods

## Description
A simple console-based Java project where users can create a virtual pet (Dog/Cat/Dragon) and interact with it.

## Features
- Feed pet
- Play with pet
- Put pet to sleep
- View status (hunger, energy, happiness)

## How to Run
1. Open in Eclipse
2. Run Main.java
