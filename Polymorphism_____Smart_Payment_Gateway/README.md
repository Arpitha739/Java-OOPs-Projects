# Smart Payment Gateway 💳

## Concepts Used
- Polymorphism
- Method Overriding
- Dynamic Binding
- Inheritance

## Description
A console-based Java payment gateway simulation demonstrating dynamic polymorphism using different payment methods.

## Features
- UPI payment simulation
- Credit card payment simulation
- Crypto payment simulation
- PayPal payment simulation
- Dynamic payment processing

## How to Run
1. Open in Eclipse
2. Run Main.java
